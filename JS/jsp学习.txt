js和jsp关系：
  js(即javascript)是一种在浏览器中运行的脚本语言，在一个网页中，CSS管理页面样式，js主要负责页面动作，而html主要负责页面内容，js部分和CSS部分经常从htm文件中单独抽离出来，前者以.js为后缀名，后者以.css为后缀名。
  jsp的中文意思是java服务页面，可以看成是html与java的结合体；jsp其实可以看做一种脚本语言，需要用servlet来编译实现（js靠浏览器解释运行），在其中可以嵌入java代码，jsp中几乎可以使用全部的java类 。jsp是java的13中核心技术规范之一，是java的一种应用情形。
  所以js和jsp就像俄罗斯和白俄罗斯一样，没有任何关系，只是恰巧名字相似而已。

简介：
JSP 技术是以 Java 语言作为脚本语言的，JSP 网页为整个服务器端的 Java 库单元(jar包?)提供了一个接口来服务于HTTP的应用程序。
SP全称Java Server Pages，是一种动态网页开发技术。它使用JSP标签在HTML网页中插入Java代码。标签通常以<%开头以%>结束。
JSP是一种Java servlet，主要用于实现Java web应用程序的用户界面部分。网页开发者们通过结合HTML代码、XHTML代码、XML元素以及嵌入JSP操作和命令来编写JSP。
JSP通过网页表单获取用户输入数据、访问数据库及其他数据源，然后动态地创建网页。
