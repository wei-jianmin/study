怎么理解反射：
反射的字面意思是光的反射，例如我要将一束光照射到墙上，除了直接照射到墙上外，也可以借助一面镜子，将光照射到镜子上，镜子把光反射到墙上。
在上面例子的基础上，在加一些要求：
如果墙上有个英语字母表，我要将光照射到不同的不同的字母上，则有两种解决方案：一是我控制将光照在不同的字母上，二是我只控制要不要发出光，并控制光准确照在镜子上，然后剩下的功能交给镜子来完成。
早期的CRT显示器及工业中用的示波器，都是采用的第二种方案。
java中的反射也跟第二种方案有一定的相似性：
调用一个类的方法或数据时，不是直接去调用特定的类，而是统一调用class类，有class负责去真正调用特定的类。通过class类，可以完成各种各样的类的调用，这里的这个class类就相当于一面镜子，这种调用机制即称为java反射。
